# Trading Bot Project - Daily Log

## 2026-02-04

### Major Redesign Complete

**Problem**: GA-based strategies showed high variance (Run 3 ✅, Run 4 ❌, Run 5 ⚠️) - overfitting issue.

**Solution Implemented**:
1. **Regime-Aware Ensemble v2** - Deterministic rules instead of learned parameters
2. **Multi-timeframe analysis** - 1h, 4h, 24h candles
3. **Three specialized strategies**:
   - Trend (EMA crossover)
   - Volatility (Bollinger Bands)
   - Mean Reversion (RSI)
4. **Consensus voting** - 2+ strategies must agree

### Key Learnings

1. **Conservative > Aggressive** - Requiring 2+ strategies to agree reduces overfitting
2. **Deterministic rules work better than GA for this timeframe** - No learned parameters to overfit
3. **Regime detection is critical** - Strategy adapts to market conditions
4. **Kraken API limits** - Only returns 721 candles regardless of request
